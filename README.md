# MyKKPlaylist

1. 建立 [KKBOX for developer](https://developer.kkbox.com/#/signin) 帳號
2. 新增一個 app 專案
   * Name 及 Description 為必填欄位
3. 將建立成功頁上 `ID` 及 `Secret` 填入 [MyKKPlaylist/build-phase/get_access_token.sh](https://github.com/vivianLiu-renyu/MyKKPlaylist/blob/master/MyKKPlaylist/build-phase/get_access_token.sh) 內
4. build app
